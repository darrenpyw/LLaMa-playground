# LLaMA-playground

A personal Agentic AI learning project using ToolAgents and local llama.cpp server with huggingface models.


## Agents

| Agent | Description |
| :---: | :--- |
| [Asynchronous Code Review Agent](./agents/async_code_review_agent.pyfilename.ext) | Asynchronous code review agent. Review source codes in a directory or by filename and output findings in markdown format. |
| [Synchronous Code Review Agent](./agents/code_review_agent.pyfilename.ext) | Synchronous code review agent. Review source codes in a directory or by filename; one at a time and output findings in markdown format. |
